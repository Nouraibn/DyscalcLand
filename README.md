# DysCalcLand

DysCalcLand is an interactive educational game that helps children with dyscalculia learn math in a fun, theme park-inspired environment. Built with **SpriteKit** and enhanced by interactive sound, it transforms math challenges into engaging adventures for kids aged 6â€“12.

## Features

- **Interactive Gameplay:** Engaging math challenges with responsive sound effects.
- **Powered by SpriteKit:** Smooth animations and immersive visuals.
- **Fun Theme Park Environment:** A playful, stimulating interface tailored for young learners.
